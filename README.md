# Python Data Types & Special Keywords

This repository contains a Python assignment covering fundamental **data types** and **special keywords** in Python.


## **üîπ Python Data Types**
Python has different built-in data types used to store values.

### **1Ô∏è‚É£ Numeric Types**
- `int` ‚Üí Whole numbers (e.g., `10`, `-5`)
- `float` ‚Üí Decimal numbers (e.g., `3.14`, `-2.5`)
- `complex` ‚Üí Numbers with imaginary parts (e.g., `2 + 3j`)

### **2Ô∏è‚É£ Sequence Types**
- `str` ‚Üí Text (e.g., `"Hello"`)
- `list` ‚Üí Ordered, **mutable** collection (e.g., `[1, 2, 3]`)
- `tuple` ‚Üí Ordered, **immutable** collection (e.g., `(10, 20, 30)`)
- `range` ‚Üí Sequence of numbers (e.g., `range(5) ‚Üí 0,1,2,3,4`)

### **3Ô∏è‚É£ Set Types**
- `set` ‚Üí Unordered, unique items (e.g., `{1, 2, 3}`)
- `frozenset` ‚Üí Immutable set (e.g., `frozenset({4, 5, 6})`)

### **4Ô∏è‚É£ Mapping Type**
- `dict` ‚Üí Key-value pairs (e.g., `{"name": "Neha", "age": 25}`)

### **5Ô∏è‚É£ Boolean & None Types**
- `bool` ‚Üí True/False values (`True`, `False`)
- `NoneType` ‚Üí Represents nothing (`None`)


## **üîπ Python Special Keywords**
Python has reserved words that **cannot** be used as variable names.

### **1Ô∏è‚É£ Control Flow Keywords**
- `if`, `elif`, `else` ‚Üí Conditional statements
- `for`, `while`, `break`, `continue` ‚Üí Looping

### **2Ô∏è‚É£ Exception Handling Keywords**
- `try`, `except`, `finally` ‚Üí Handling errors
- `raise` ‚Üí Generate custom errors

### **3Ô∏è‚É£ Function & Class Keywords**
- `def` ‚Üí Defines a function
- `class` ‚Üí Defines a class
- `return` ‚Üí Returns a value from a function

# Google Colab: Introduction and Overview

## Description

Google Colab (short for Colaboratory) is a free, cloud-based platform provided by Google that allows you to write and execute Python code in a Jupyter notebook environment. It‚Äôs widely used for machine learning, data analysis, and research, as it offers the ability to run Python code on Google‚Äôs cloud infrastructure without requiring any installation.

## Key Features

- **Cloud-based**: No installation required, you can run notebooks directly in the cloud.
- **Free Access to GPUs/TPUs**: It provides access to hardware accelerators (like GPUs and TPUs) for free, which is very useful for deep learning projects.
- **Jupyter Notebook Interface**: Google Colab is built on top of the Jupyter notebook, allowing you to combine code, visualizations, and markdown text in one document.
- **Collaboration**: You can share your notebooks with others, similar to Google Docs, and collaborate in real time.
- **Integration with Google Drive**: You can save, open, and share your notebooks directly from your Google Drive account.
- **Easy to Use**: It‚Äôs beginner-friendly and doesn‚Äôt require complex setup or local resources.

## How to Use Google Colab

1. **Access**: Go to Google Colab and sign in with your Google account.
2. **Create a New Notebook**: You can start a new notebook by clicking "File" ‚Üí "New Notebook."
3. **Write Code**: Write Python code in the cells and press Shift + Enter to execute it.
4. **Add Text**: You can add descriptive text using markdown by selecting a cell type as "Text."
5. **Save Notebooks**: Colab automatically saves your notebooks to Google Drive.
6. **Share**: Use the "Share" button in the top-right corner to share your notebooks with others.
